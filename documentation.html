Introducción

Este tutorial le permitirá implementar una aplicación Java en minutos.
En este tutorial asumimos que ya tiene:

  • Cuenta en Pakay
  • Java 8 instalado
  • Maven instalado

Código

Esto declara un único tipo de proceso, web y el comando necesario para ejecutarlo. El nombre web es importante aquí. Declara que este tipo de proceso se adjuntará a la pila de enrutamiento HTTP de Pakay y podrá recibir tráfico web. Los archivos de proceso pueden contener tipos de procesos adicionales. Por ejemplo, puede declarar uno para un trabajador en segundo plano que procesa elementos de una cola.
