PAKAY

Pakay es una plataforma en la nube basada en contenedores completamente administrada, con servicios de datos integrados y un poderoso ecosistema, para implementar y ejecutar aplicaciones modernas. La experiencia del desarrollador de Pakay se basa en un enfoque centrado en la aplicación para la entrega de software y se integra con los flujos de trabajo y las herramientas de desarrollo más populares de la actualidad.

MÁS PRODUCTOS

Pakay Platform

La plataforma Pakay ejecuta sus aplicaciones dentro de dynos: contenedores inteligentes que se ejecutan en un entorno de tiempo de ejecución confiable completamente administrado. Puede implementar código escrito en Node, Ruby, Java, PHP, Python, Go, Scala o Clojure o usar un paquete de compilación personalizado para implementar aplicaciones en otro idioma que elija. Y puede escalar su aplicación al instante aumentando la cantidad y el tipo de dinamómetro en el que se ejecuta su aplicación.

Pakay Postgres

La base de datos SQL Postgres confiable, segura y escalable de Pakay como servicio, optimizada para desarrolladores. Con características como protección continua, reversión y alta disponibilidad, Pakay Postgres garantiza que sus datos estén seguros y siempre disponibles a escala. Pakay Postgres es fácil de configurar y proporciona potentes funciones como bifurcaciones, seguidores y clips de datos, para que pueda concentrarse en crear las aplicaciones que hacen crecer su negocio, no en la gestión de bases de datos.

Pakay Redis

Pakay Redis le brinda el Redis que ama con una experiencia de desarrollador más sólida, completamente administrada y entregada como un servicio por los expertos operativos de Pakay. Pakay Redis le permite obtener visibilidad del rendimiento, administrar sus instancias con una CLI potente y federar datos con Postgres para obtener información empresarial mediante herramientas SQL.

Apache kafka en Pakay

Apache Kafka en Pakay es un servicio de mensajería distribuida que combina la solución líder de código abierto para administrar flujos de eventos con la experiencia de desarrollador de Pakay, para que pueda crear aplicaciones con uso intensivo de datos con facilidad.
